# Abhishek Bansal - Portfolio Website

A professional, responsive portfolio website for Abhishek Bansal — B.Tech Data Science Student at MIET Meerut (Batch 2027), Aspiring Data Analyst.

## 📁 File Structure

```
portfolio_abhishek/
├── index.html     → Main HTML structure & content
├── style.css      → All styles, animations, layout
├── script.js      → Interactivity, animations, form handling
└── README.md      → This file
```

## 🚀 How to Use

1. Extract the ZIP folder
2. Open `index.html` in any modern web browser
3. No installation or backend setup needed!

## ✨ Features

- **Responsive Design** — Works on desktop, tablet, and mobile
- **Smooth Animations** — Scroll reveal, typing effect, skill bars
- **Professional Theme** — Navy blue + gold corporate palette
- **5 Sections** — Hero, About, Skills, Projects, Education, Contact
- **Typing Role Animation** — Cycles through career roles
- **Animated Skill Bars** — Triggered on scroll
- **Contact Form** — Simulated submission with success message
- **Hamburger Menu** — Mobile-friendly navigation

## 🛠 Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Playfair Display + DM Sans)
- Font Awesome Icons
